# Opacity

Opacity tokens provide a consistent scale from fully transparent (0) to fully opaque (100). Source: `tokens/primitives/opacity.json`.

## CSS usage

```css
.overlay {
  background: var(--color-black);
  opacity: var(--opacity-50);  /* 0.5 */
}

.disabled {
  opacity: var(--opacity-80);
  pointer-events: none;
}

.ghost {
  opacity: var(--opacity-0);
}
```
